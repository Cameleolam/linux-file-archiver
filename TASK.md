# File Archiving Script for Linux Systems

## 1. Task
Implement a Python script to archive files owned by members of a specified group on a Linux system.
The script should take the group name as a parameter and move files from user home directories to a configurable archive folder, preserving the original directory structure.

## 2.Requirements
- The program should be robust, e.g. with respect to multiple invocations in short time. - The events and results should be available in a log file.
- The program should be installable as a Debian package and should be made available for download.
